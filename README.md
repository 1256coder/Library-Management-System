# 📚 Library Management System

A modern and interactive **Library Management System** developed using **HTML, CSS, and JavaScript**. The project provides an intuitive interface for managing books, users, and borrowing activities while simulating the core functionalities of a real-world library.

---

## 🚀 Features

### 👤 User Features
- User Registration & Login
- User Profile Dashboard
- Browse Library Collection
- Search Books
  - By Title
  - By Author
  - By ISBN
- Browse Books by Categories
- Add Books to Borrow Cart
- Borrow Multiple Books
- Return Borrowed Books
- Borrowing History
- Fine Management System

### 📖 Book Management
- View Complete Book Collection
- Book Categories
- Book Availability Status
- Book Details
- Book Cover Images
- Book Descriptions
- ISBN Tracking

### 🛠️ Admin Features
- Admin Login
- Admin Dashboard
- Add New Books
- Edit Existing Books
- Delete Books
- Manage Users
- View Library Statistics
- Track Available Books

### 🔍 Search System
- Search by Book Title
- Search by Author
- Search by ISBN
- Category Filtering

### 🎨 User Interface
- Modern Responsive Design
- Interactive Book Cards
- Dark Theme
- Animated Modals
- Responsive Layout
- Clean Navigation
